# AI Tutor - Modular Monolith AI Solution

Hệ thống AI Tutor được xây dựng trên nền tảng .NET 9, sử dụng kiến trúc Modular Monolith và Clean Architecture. Dự án tích hợp các công cụ AI mạnh mẽ (Ollama, Semantic Kernel, Qdrant) để mang lại trải nghiệm học tập cá nhân hóa.

## Đặc điểm nổi bật
- **RAG Pipeline**: Xử lý kiến thức từ tài liệu PDF cá nhân.
- **Socratic Method**: Phương pháp giảng dạy gợi mở từ AI.
- **Real-time Streaming**: Phản hồi tức thì qua SignalR.
- **Local First**: Chạy hoàn toàn cục bộ hoặc tích hợp Cloud qua API.
